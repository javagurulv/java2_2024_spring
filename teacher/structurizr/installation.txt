
Скачать structurizr-lite.war по ссылке https://github.com/structurizr/lite/releases

В пакет teacher/structurizr установить файл structurizr-lite.war.

Класс запуска StructurizrStarter для structurizr-lite.war будет находиться teacher/step_*/src/main/java/diagram
если в этом шаге предусмотрена диаграмма. Соответственно, при запуске будет отображаться диаграмма на уровне выполненных заданий конкретного шага.
В следующем шаге диаграмма дополняется.

Запуск StructurizrStarter разворачивает structurizr-lite на порт 8080, открытие браузера
по ссылке, выведенной на консоль. Разворачивание диаграмм в браузере занимает время до одной / двух минут.

Для очистки порта 8080 запускать main в классе PortUtil в teacher/step_*/src/main/java/diagram. Запуск StructurizrStarter так же запускает очистку порта, поэтому запускать PortUtil
при переходе в диаграммы другого шага нет необходимости.
